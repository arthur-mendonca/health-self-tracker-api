## Regras do projeto

- Este projeto trata somente de backend. O frontend é um projeto separado e se encontra em outro repositório.
- Nunca instale dependências sem avisar antes.
- Criar um arquivo na raiz listando os comandos para rodar o app (localmente e em produção) e como rodar testes (quando cabível). Ex: `RUNNING.md` ou `COMMANDS.md`.
- Criar e manter atualizado, sempre que alguma nova rota for adicionada, um arquivo .json com a coleção de HTTP Requests para ser usada no Postman/Insomnia (ex: `requests.json`).
